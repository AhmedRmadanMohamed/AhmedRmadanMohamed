# Setup

1. Copy `README.md` to the root of your GitHub profile repository.
2. Copy the full `assets` folder beside it.
3. Add your existing animated profile image as `assets/profile.gif`.
4. The repository name must exactly match your GitHub username: `AhmedRmadanMohamed`.
5. Commit and push the files.

## Notes

- GitHub controls the normal body font of README files. The custom typography in this design is implemented inside the SVG banner assets and the animated headline.
- Images use percentage-based or naturally scaling widths, and wide fixed tables were intentionally avoided for mobile compatibility.
- External image services can occasionally be rate-limited. The local SVG banners remain available because they are stored in the repository.
